Abstract: Foi necessário converter o dataset XML para json. Após essa conversão, foi necessário transformá-lo num jsonArray. Para importar o dataset (que se encontra na diretoria data) para mongoDB será apenas necessário correr o comando "mongoimport -d obras -c obras --jsonArray obras.json". Depois apenas é necessário instalar os pacotes com "npm i --save" e "npm i mongoose --save". Para correr o servidor, apenas é necessário fazer "npm start". Para o Exercicio 2, basta apenas instalar os pacotes e também o axios com "npm i axios --save" e correr com "npm start". A pergunta 1 do exercicio 2 encontra-se respondida no ficheiro com o nome "Queries.txt".
